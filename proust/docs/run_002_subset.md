# Run-002 Subset

## Goal

`run-002` should extend the first reviewed benchmark without leaving its local terrain.

The batch therefore includes:

- the 3 reviewed benchmark units from `run-001`
- 7 additional units from `v1-p1-combray`
- only material that still works with the current conservative alias map

This keeps the second run close enough to `run-001` for calibration while adding more variation in:

- narrated prestige
- social misrecognition
- aristocratic mediation
- exposed snobbery
- evasive self-positioning
- narrated social decline

## Included benchmark units

- `v1-p1-combray#p-17`
- `v1-p1-combray#p-274-p-275`
- `v1-p1-combray#p-312-p-313`

These remain in the batch so later candidate annotations can be compared against the reviewed benchmark directly.

## New units

### `v1-p1-combray#p-20`

Why:

- compact Swann passage
- strong family misreading of elite association
- good test of narrated prestige vs. bourgeois interpretive limits

Core dynamic:

- Swann's real social proximity to a princess is locally discounted by the family's mental model of rank.

### `v1-p1-combray#p-21-p-22`

Why:

- explicit local treatment of Swann as socially useful but insufficiently prestigious
- clear distinction between practical reliance and honor

Core dynamic:

- the family uses Swann instrumentally while withholding full social esteem.

### `v1-p1-combray#p-23-p-24`

Why:

- brings `Mme de Villeparisis` into the frame
- shows prestige transfer behaving perversely

Core dynamic:

- learning that `Mme de Villeparisis` values Swann does not raise Swann for the aunt; instead it lowers `Mme de Villeparisis`.

### `v1-p1-combray#p-25-p-26`

Why:

- explicit external confirmation of Swann's status
- strong bridge between newspaper prestige and family recognition

Core dynamic:

- Swann's standing is partially legible only when ratified by aristocratic and cultural signals.

### `v1-p1-combray#p-278-p-279`

Why:

- compact Legrandin unit
- socially revealing bodily and rhetorical evasion

Core dynamic:

- when asked about Balbec connections, Legrandin signals knowledge and concern while trying not to answer plainly.

### `v1-p1-combray#p-283-p-284`

Why:

- second Legrandin pressure test
- good contrast between direct questioning and ornamental self-protection

Core dynamic:

- Legrandin refuses ordinary social disclosure by retreating into evasive abstraction.

### `v1-p1-combray#p-310-p-311`

Why:

- extends the Vinteuil cluster backward from the benchmark unit
- useful test of narrated diminishment and pity without requiring new aliases

Core dynamic:

- Vinteuil is presented as socially broken down by grief, avoidance, and local shame.

## Shape of the batch

Total units: `10`

Breakdown:

- `3` reviewed benchmark units
- `7` new candidate units

## Practical use

`run-002` is meant to be the first expansion batch.

It is not yet a reviewed benchmark.

Its purpose is to let us:

- generate a slightly larger candidate run
- keep overlap with `run-001`
- compare shared reviewed units directly
- review the 7 new units incrementally
